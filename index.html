glTF View João Sousa 2021

Este modulo carrega um ficheiro 3D (glTF ou glb) e adiciona ao canvas um cursor "grabbing", uma sombra no modelo (opcional), e cria um "RoomEnvironment". Tudo com o auxílio do Three.js.

Basta definir uma tag com a classe de glTFView e adicionar as propriedades.

Atributos:

  • glTFfile="<caminho>": caminho para o ficheiro 3d.
  • auto: se presente o ficheiro roda automaticamente.
  • w="<int>": largura do canvas, se omitir é redimensionado de acordo com a largura da janela.
  • h=<int>": altura do canvas.
  • zoom: se presente, permite utilizar o zoom com o mouse wheel.
  • transparent: se presente, o fundo do canvas fica transparente mas não é possível adicionar a sombra.
  • bg="<#hex>": define a cor do background (esta opção é ignorada quando usado a opção "transparent"), e quando não é definida por omissão a cor de background é branco.
  • scale="<float>": define o tamanho do objecto. (permite redimensinar caso não apareça na camera)
  • ld="<#hex>": define a cor da borda do loading

Exemplo:

<div class="glTFView" glTFfile="models/model.gltf" auto zoom w="300" h="300"></div>
